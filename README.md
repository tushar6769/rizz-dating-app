# ⚡ Rizz AI - Intelligent Dating Assistant

An AI-powered "Rizz" coaching and dating assistant application that turns dry conversations and raw chat screenshots into smooth, witty, and context-aware responses. Powered by the Google AI Studio Gemini API, this full-stack application helps users elevate their conversational game.

## 🚀 Features

* **AI Rizz Generator:** Input a prompt, profile bio, or recent chat message to generate tailored icebreakers and replies.
* **Tone Selection:** Choose your vibe—Bold & Witty, Smooth & Direct, or Chill & Casual.
* **Full-Stack Architecture:** Built with a fast Vite + TypeScript frontend and a robust Node.js backend server.
* **Real-time Analytics Dashboard:** Track conversation success metrics and performance trends.

## 🛠️ Tech Stack

* **Frontend:** React, Vite, TypeScript, Tailwind CSS
* **Backend:** Node.js, Express, TypeScript
* **AI Integration:** Google AI Studio (Gemini API)
* **Data Visualization:** Power BI layouts for tracking engagement metrics
